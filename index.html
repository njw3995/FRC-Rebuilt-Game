GAME CONTROLS

Keyboard Player 1

W A S D = Move

J / L = Turn

Space = Shoot/Pass

E = Toggle Blitz intake side

Keyboard Player 2

Arrows = Move

, / . = Turn

Enter = Shoot/Pass

Right Shift or / = Toggle Blitz intake side

Controller

Left Joystick = Move

Right Joystick = Turn

A, Right Bumper, or Right Trigger = Shoot/Pass

Left Trigger = Toggle Blitz intake side

Blitz Robot

The player toggle input changes which side the intake is on.

P1 keyboard uses E. P2 keyboard uses Right Shift or /. Controller uses Left Trigger.
