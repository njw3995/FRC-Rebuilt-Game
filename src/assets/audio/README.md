# Audio assets

The game loads cue files from this folder through `src/scripts/audio.js`.

Current files:

```text
auto-start.wav
teleop-start.wav
shift-change.wav
endgame-start.wav
end.wav
```

To restore the original audio exactly, replace these files with the original cue files and keep the same filenames, or update `AUDIO_SOURCES` in `src/scripts/audio.js`.
